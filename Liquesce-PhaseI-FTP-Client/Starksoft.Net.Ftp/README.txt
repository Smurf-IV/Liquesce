README.txt

The Starksoft FTP Code Library is a software library built on the Microsoft .NET 2.0 Framework and coded in 100% 
C# managed code.  This library enables C# and VB.NET users to communicate with almost any FTP server in several 
modes of operation.  This library supports SSL connection, connections through a proxy, passive and active mode
connections, asynchronous calls, zlib compression, and many other features.

http://www.starksoft.com for the latest version


-------------------------------------------------------------------------------------------------------------------
REQUIREMENTS

    * Microsoft .NET Framework version 2.0 or higher.
