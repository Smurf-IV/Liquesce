Different Modes:

  • Folder
  • Priority
  • Balanced

The Folder Tray form is mentioned a lot, but it also includes another form that allows you to see where a file or directory may be placed across the merged drives.

 

Folder Mode

  • Attempts to keep all new creations in the directory they are targeted at.
  • Once the “HoldOffBytes” value is breached, then it will create a new directory with the same path elements on the highest drive that has greater than the “HoldOffBytes”.
  • The “HoldOffBytes” must be larger than the largest file that the user envisages creating. so if Archiving home videos @ DVD size of 4.4GB, then make the Hold of Size 5GB as a minimum.
  • Currently updates to existing files that cause a breach in the space, will not be moved to another drive.
  • The Folder Tray form, will show the “HoldOffBytes” status in different colours: Orange for getting close, and Red for breached.

 

Priority

  • All new Creations will be in the order the drives have been placed in the management app.
  • Once the “HoldOffBytes” value is breached, then it will create a new directory with the same path elements on the highest drive that has greater than the “HoldOffBytes”.
  • This mode is good if you want all new creations to be on a fast drive, that will then allow a backup application to move them to slower drives in Idle time etc..
  • The Folder Tray form, will show the next drive to be used with a green colour.

 

Balanced

  • It will place the files to get a balanced free space on all disks.
  • This will mean that the bigger (and usually faster) disks will be used first.
  • There may also be a speed improvements due to parallel disk use, like in a software RAID0 but this will only be faster if you are copying multiple files at the same time. If you only copy one file after another it will only access one disk at a time...
  • The Folder Tray form, will show the amount of free space, either in proportional or default windows explorer styles.
